# Croissant

## Install
- Node.js 17.9
    1. `npm install`
- Python 3.9
    1. `python -m venv venv`
    2. Mac or Linux: `. venv/bin/activate`, or Windows: `venv\Scripts\activate`
    4. `pip install -r requirements.txt`

## Run Server
- Front end (develop)
    1. `cd spa`
    2. `npm run dev`
- Front end (start)
    1. `cd spa`
    2. `npm run build`
    3. `npm start`
- Back end
    1. `cd api`
    2. `python manage.py runserver`
